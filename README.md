# Binions

## SpingFramework-Project-예랑예신(OneStop Wedding info communiuty)
SpringFramework를 활용한 결혼 정보 커뮤니티

## 🖥️ 프로젝트 소개

결혼을 준비하는 사용자부터 경험자까지
익명성을 바탕으로 다양한 정보를 공유하는 플랫폼 제공
